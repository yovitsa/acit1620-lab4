CSS Transitions

CSS transitions provide a way to control animation speed when changing CSS properties

You can cause the changes in a property to take place over a period of time.

Transitions enable you to define the transition between two states of an element.

Different states may be defined using pseudo-classes like :hover or :active or dynamically set using JavaScript.

CSS Variables

Custom properties (sometimes referred to as CSS variables or cascading variables)

They are entities defined by CSS authors that represent specific values to be reused throughout a document.

CSS variables are declarations that start with a double dash (--) and have a name and a value.

It is also possible to dynamically change CSS variables.
